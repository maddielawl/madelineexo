Description : 
Ce projet se veux être une compilation d'outils plus ou moins utiles
En lançant le projet, on nous demande quel programme utiliser
Le premier est une calculette simple pour faire des oppérations entre 2 chiffres
Le deuxième est un outil qui enlève tous les accents d'une chaine de caractères
Le 3ème permet de choisir aléatoirement une classe qui sont définies dans un énum
Et enfin le dernier permets de répéter un string qui est défini de base.

Il existe un test unitaire pour la répétition du string et pour la séléction de classe.

2 githooks sont installés respectivement pour le pre commit et le message de commit
Le pre commit verifie si on indexe bien toutes les modifications
Le hook de message de commit verifie bien si on as mis le nom des scripts a l'interieur du message.

Par soucis de deadline et le fait que j'ai été seule, j'ai mal pris connaisance de la consigne et ils ont été installés a la fin.
Après avoir vu avec vous, j'ai demandé l'aide de Moeva, un 5ème année, qui m'as expliqué des choses en rapport avec l'exercise, il m'as aidé environ 2 heures pour setup le tout est a conceptualisé le script "Select Class"
